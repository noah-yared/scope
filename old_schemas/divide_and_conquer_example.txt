1. PROBLEM HEADER
- algorithm: Maximum Subarray (Kadane's Algorithm)
- input: [-2, 1, -3, 4, -1, 2, 1, -5, 4]
- goal: Find the maximum-sum contiguous subarray and return its sum.

2. BASE CASE
If the array has size 0:
    return 0
If the array has size 1:
    return the single element

3. INITIALIZE (Kadane-specific)
current_sum = input[0]
best_sum = input[0]

4. ITERATE
For i from 1 to n-1:
    current_sum = max(input[i], current_sum + input[i])
    best_sum = max(best_sum, current_sum)

5. FINAL RESULT
final_result = best_sum
